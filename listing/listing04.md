Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Канал ch никогда не закрывается, и поэтому основная горутина бесконечно ожидает значения из канала, даже после того, как все значения были отправлены.

Цикл for n := range ch продолжает работать, пока канал не будет закрыт, поэтому получается бесконечный цикл - канал не закрывается.

```

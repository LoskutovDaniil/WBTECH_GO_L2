Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Выведет срез элементов, начиная с 1 индекса и заканчивая 3 индексом включительно, то есть 77, 78, 79

```

Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Функция test(): вернет значение 2
Почему:
x объявляется как именованный возвращаемый результат (x int).
Перед фактическим возвращением значения, выполняется отложенная функция (defer), которая увеличивает x на 1 (x++).

Функция anotherTest(): вернет значение 1
Почему:
x объявляется как локальная переменная
Перед фактическим возвращением значения, выполняется отложенная функция (defer), которая увеличивает x на 1 (x++).
Однако, x в отложенной функции — это локальная переменная, а не возвращаемое значение, поэтому это не влияет на уже подготовленное возвращаемое значение.

```
